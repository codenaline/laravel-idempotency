# Security Policy

## Supported Versions
We support the latest stable release of this package.  
Older versions may not receive security updates.

## Reporting a Vulnerability
If you discover a security vulnerability in this package:

- **Do not open a public issue.**
- Please report it privately by emailing: mahdirezaei.dev@gmail.com  
- Include a clear description, affected version, and steps to reproduce.

We will review the report and respond as quickly as possible.
